Професійний курс Digital Design

Група для підлітків 13-16 років 12 місяців 2 заняття на тиждень по 60-90 хв

Повноцінна навчальна програма з дизайну сайтів, застосунків, ігор та реклами від школи GoIT, адаптована для підлітків. На курсі ми готуємо молодших спеціалістів з цифрового дизайну для роботи в комерційних проєктах.

На цьому курсі студенти навчаються та працюють в таких програмах:
  • Adobe Photoshop

    Найпопулярніший редактор растрових зображень від компанії Adobe

  • Adobe Lightroom

    Найкращий графічний редактор Adobe для роботи з цифровими фотографіями

  • Adobe Illustrator

    Програма компанії Adobe, яка створена для роботи з векторної графікою

  • Adobe After Effect

    Програма, в якій ваша дитини зробить свій перший анімаційний ролик

  • Figma

    Міжнародна платформа, яка дозволяє конструювати сайти без знання коду
